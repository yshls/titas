# TiTaS (Tiki-Taka Script)

> "요즘엔 AI가 있는데 굳이?" 라는 질문에 답하며,
> AI는 줄 수 없는 '실전 영어 회화 근육'을 만드는 훈련장입니다.

## 1. 프로젝트 소개 (Introduction)

TiTaS는 AI 챗봇처럼 '완벽한 정답'을 알려주는 수동적 학습이 아닌, '정해진 대본(Script)'을 기반으로 사용자가 직접 말하며 부딪혀보는 '능동적 훈련(Tiki-Taka)'을 위해 태어났습니다.

언어 학습은 정답을 얻는 과정이 아니라, 스스로 틀려보고 교정하며 '근육을 만드는 과정'이라는 철학에서 시작되었습니다.

## 2. 핵심 문제 정의 (The Problem)

기존 AI 기반 영어 학습 도구들은 다음과 같은 3가지 명확한 한계를 가집니다.

1.  **'정해진 대본'의 부재:** AI는 자유 대화(Free-talking)에는 강하지만, 특정 상황이나 표현을 반복적으로 연습하는 '롤플레잉(Tiki-Taka)' 경험을 제공하지 못합니다.
2.  **모호한 피드백:** "Good job!" 이상의 실질적인 피드백이 부족합니다. 사소한 발음, 실수(정답/오답)를 실시간으로 명확하게 채점받기 어렵습니다.
3.  **데이터의 휘발성:** AI와의 대화는 누적/분석되지 않습니다. 내가 무엇을 자주 틀리는지, 약점이 무엇인지 데이터 기반의 '개인화 대시보드'를 제공받지 못합니다.

<br>

## 3. 주요 기능 (Functional Requirements)

TiTaS는 위 4가지 문제를 해결하기 위해 다음과 같은 핵심 기능을 제공합니다.

### 1. 대본 생성 및 관리 (Script Management)

- **맞춤 대본 생성**: 사용자가 직접 대본의 제목과 내용을 작성할 수 있습니다.
- **다중 화자 설정**: 'Speaker 1', 'Speaker 2', 'Speaker3' 3명의 화자를 설정하고, 각 화자별로 대사를 입력할 수 있습니다.
- **실시간 대본 편집**: 추가한 대사를 즉시 삭제하며 대본을 완성할 수 있습니다.
- **대본 목록 및 정렬**: 생성된 모든 대본을 목록에서 확인하고, 최신순, 오래된순, 제목순으로 정렬할 수 있습니다.

### 2. 역할극 기반 말하기 연습 (Role-Play Practice)

- **역할 선택**: 연습을 시작하기 전, 사용자가 직접 연기할 역할을 선택하여 몰입감을 높입니다
- **자동 대사 읽기 (TTS)**: 사용자가 선택하지 않은 역할의 대사는 시스템이 자동으로 읽어주어 실제 대화처럼 연습할 수 있습니다.
- **실시간 발음 피드백**:
  - **음성/키보드 입력**: 마이크를 사용한 음성 입력과 키보드 입력 두 가지 모드를 모두 지원합니다.
  - **단어 비교 분석**: 사용자의 발음을 원래 대사와 단어 단위로 비교하여, 정확한 단어, 틀린 단어, 빠뜨린 단어를 색상으로 명확하게 구분하여 보여줍니다.
- **연습 제어**: 연습 중 힌트 보기, 재시도, 저장 후 종료 등 다양한 제어 기능을 제공합니다.

### 3. 학습 성과 분석 및 리뷰 (Progress Analysis)

- **대시보드**:
  - **연습 캘린더**: 어떤 날에 연습을 했는지 한눈에 볼 수 있는 캘린더를 제공합니다.
  - **핵심 통계**: 평균 정확도, 총 대본 수, 총 대사 라인 수 등 주요 성과 지표를 시각화하여 보여줍니다.
- **약점 분석 (Weak Spots Review)**:
  - **자주 틀리는 단어 목록**: 모든 연습 기록을 분석하여 가장 많이 틀리는 단어 Top 10을 보여줍니다.
  - **발음 듣기**: 목록에 있는 단어의 정확한 발음을 바로 들어볼 수 있습니다.

### 4. 사용자 경험 (User Experience)

- **데이터 영속성**: 생성한 모든 대본과 연습 기록은 브라우저의 로컬 스토리지에 저장되어, 언제든지 이어서 학습할 수 있습니다.
- **반응형 디자인**: 데스크톱, 태블릿, 모바일 등 모든 기기에서 최적화된 UI를 제공합니다.

## 6. 시작하기 (Getting Started)

### 설치 (Installation)

```bash
# 1. 저장소를 복제합니다.
git clone https://github.com/yshls/titas.git

# 2. 프로젝트 폴더로 이동합니다.
cd titas

# 3. 의존성 패키지를 설치합니다.
npm install

# 4. 개발 서버를 실행합니다.
npm run dev

```
